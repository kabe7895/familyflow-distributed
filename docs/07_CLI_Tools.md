# Development Tools

This document describes the main development tools used during the implementation of the FamilyFlow application.


---

# Visual Studio Code

The project was developed using **Visual Studio Code**.

Used for:

- Java development
- Spring Boot backend development
- React frontend development
- Markdown documentation
- Git management
- Project structure management


---

# OpenAI Codex CLI

The **OpenAI Codex CLI** was installed and tested as a command line coding assistant.

Purpose:

- Code analysis
- Assistance during implementation
- Understanding existing project structures
- Supporting development tasks directly in the terminal


Example usage:

```text
Analyze project structure

Review source code

Assist with implementation steps
```


---

# Google Gemini CLI

The **Google Gemini CLI** was installed and evaluated as an additional AI-based development assistant.

Purpose:

- Code understanding
- Alternative AI-assisted analysis
- Supporting software development tasks
- Comparing different AI coding assistants


Example usage:

```text
Analyze source files

Explain implementation details

Support development decisions
```


---

# Maven

Maven was used as the build and dependency management system for the Spring Boot applications.

Used commands:

```bash
mvn clean test
```

```bash
mvn spring-boot:run
```

Responsibilities:

- Managing dependencies
- Building the application
- Running tests
- Starting Spring Boot services


---

# PostgreSQL and pgAdmin

PostgreSQL was used as the persistent database system.

pgAdmin was used for:

- Database administration
- Inspecting tables
- Verifying stored data
- Managing the FamilyFlow database


Database:

```text
familyflow
```
